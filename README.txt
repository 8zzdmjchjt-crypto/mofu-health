小動物健康手帳 PWA試作版

これはiPhone/Android/PCのブラウザで動く「アプリ風」試作版です。
iPhoneではSafariで開いて「ホーム画面に追加」すると、アプリのように起動できます。

使い方:
1. ZIPを解凍
2. index.htmlを開く
3. iPhoneの場合はSafariで開き、共有→ホーム画面に追加

できること:
- 複数ペット登録
- 日付ごとの記録
- 体重、ごはん、食欲、元気、うんち
- 砂浴び、砂浴び時間
- 部屋んぽ、部屋んぽ時間
- 温度、湿度、メモ
- カレンダー表示
- 体重グラフ
- データ書き出し

注意:
- App Storeに出す正式アプリではなく、動く試作版です。
- データは端末ブラウザ内に保存されます。
- 診断や治療の代わりにはなりません。
